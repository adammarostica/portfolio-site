PROJECTS

Walky Talky

Looking to fill the next few hours with podcasts? Walky Talky uses custom React hooks, the Listen Notes API, and a recursive algorithm to offer you a podcast playlist that suits your interests and lasts just as long as you need it to. You can even re-order your playlists, save them for later consumption, and listen to them in a custom audio player.

Built with:

  • React

  • Firebase

  • Rest API

  • CSS3

  • Git

Poll in the Wall

Create polls for others to vote on using this stylish and responsive app built with React, React Router, Firebase, and Local Storage. Users can specify a custom path where they would like others to find their poll, and once the user has voted, the app marks their browser so they only see the results if they visit the poll again.
